# Hans Sandberg's personal site

https://hanssandberg.com/

This is the source code for my personal website, built with [Astro](https://astro.build/).

It showcases my experience and skills as a designer and front-end developer, including case studies.

Feel free to explore, learn from, or reuse small pieces of code. Please don’t copy the design or content directly.

## Getting started

- Install dependencies: `npm install`
- Start local dev server (http://localhost:4321/): `npm run dev`
- Trigger a production build: `npm run build`
